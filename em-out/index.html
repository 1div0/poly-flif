About this demo

This is a playground for trying truncation for the new FLIF image format. It is also an early demo of a possible browser polyfill for the format.

About FLIF

FLIF is a lossless image format based on MANIAC compression. MANIAC (Meta-Adaptive Near-zero Integer Arithmetic Coding) is a variant of CABAC (context-adaptive binary arithmetic coding), where the contexts are nodes of decision trees which are dynamically learned at encode time.

FLIF outperforms PNG, FFV1, lossless WebP, lossless BPG and lossless JPEG2000 in terms of compression ratio.

Note that the FLIF spec is not yet defined completely. And the code is not optimised yet.

Do not use this in production yet.
